# Getting Started

Create your own notebook exactly like this one and customize it to yoru needs. Astropads are speedy to setup, free, and 100% open source.

- **Easy to use** - add a new Markdown or MDX page and it appears magically in your navigation
- **Writing focus** - beautiful typography and a minimalist design managed with easy-to-use Markdown and MDX
- **Speedy and accessible** - static files, responsive system fonts, minimal JavaScript, and full screen reader support
- **SEO ready** - meta tags for SEO and social sharing out of the box, [JSON-LD](https://json-ld.org/) for more dynamic search results

Astropads are modern notebooks you can use to create documentation, sketch out new ideas, or plan a new project.
